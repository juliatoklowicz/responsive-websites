BLOK 2

HTML (ang. HyperText Markup Language) - hipertekstowy język znaczników, podstawowy język tworzenia stron internetowych.
Pierwsza, publicznie dostępna, specyfikacja języka HTML, nazwana HTML Tags, została zamieszczona w Internecie przez Bernersa-Lee w 1991. Zawierała 22 znaczniki, tworzące początkowy, prosty szkielet HTML-a.

BLOK 3

Kaskadowe arkusze stylów (ang. Cascading Style Sheets, CSS) to język służący do opisu formy prezentacji (wyświetlania) stron WWW.
CSS został opracowany przez organizację W3C (World Wide Web Consortium) w 1996 r.
